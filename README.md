# HardOff Searcher MVP

Next.js で作成した、スマホ縦画面向けのフロントエンド試作です。

## 画面構成
- 入力画面
- 解析中画面
- 結果画面（候補カード 5 件）

## セットアップ
```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。
